# Accessibility

> Summary page. Canonical source: docs/ACCESSIBILITY.md and docs/A11Y-CONTRAST.md.

## Conformance target

Agterhuis.Ui targets WCAG 2.2 AA.

## Evidence model

- Documented contrast matrices for text and non-text UI boundaries.
- Keyboard and focus behavior checks in demo and wrappers.
- Automated checks via Playwright + axe route sweeps.
- Guard tests that fail builds when accessibility contracts regress.
