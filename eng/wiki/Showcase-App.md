# Showcase App

> Summary page. Canonical source: docs/SHOWCASE-COVERAGE.md.

The sample showcase app demonstrates a realistic workflow shell for field-service style scenarios:

- Werkorders dashboard and detail flow
- Planning scheduler board
- Customer/project/asset routes
- Servicedesk and reporting views

It is used for visual QA, route smoke coverage, and documentation screenshots.
