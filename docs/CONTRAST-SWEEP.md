## Notes

- Disabled elements are tracked separately. They are exempt from SC 1.4.3, but the sweep still flags anything below a 2:1 floor as onwaarneembaar.
- Gradient and glass backgrounds are evaluated from their rendered color stops, with the worst sampled stop used for reporting.
- State checks run on representative buttons, navigation, and input routes with hover/focus where the page exposes a cheap interaction target.
